# onboarding

A new Flutter project.

## Getting Started

**Step 1:**

Download or clone this repo by using the link below:

```
https://github.com/kelpo123/onboarding.git
```

**Step 2:**

Go to directory folder

```
cd sampleApp/
```

**Step 3:**

Check Emulator List

```
flutter emulator
```

**Step 4:**

Connecting To Emulator

```
flutter emulator --launch <emulator id>
```

**Step 5:**

Running The Project

```
flutter run
```
